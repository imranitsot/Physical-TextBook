# References

## Official Documentation
*   **[ROS 2 Documentation (Humble)](https://docs.ros.org/en/humble/)**: The bible for all things ROS 2.
*   **[Gazebo Sim](https://gazebosim.org/docs)**: Guides for ignition/harmonic simulation.
*   **[NVIDIA Isaac Sim](https://docs.nvidia.com/isaac/isaac_sim/index.html)**: High-fidelity simulation on Omniverse.
*   **[MoveIt 2](https://moveit.picknik.ai/)**: The motion planning framework for ROS 2.

*   **Sim-to-Real**: [Solving Rubik's Cube with a Robot Hand](https://openai.com/research/solving-rubiks-cube) (OpenAI - for Domain Randomization details).
