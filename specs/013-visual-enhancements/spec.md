# Spec 013: Visual Enhancements

## 1. Background
Text-heavy documentation is boring. "Physical AI" is a highly visual field. The textbook needed high-quality custom visuals to illustrate concepts.

## 2. Requirements
*   **Custom Images**: No stock photos. Content-specific generations (e.g., "ROS 2 Wireframe", "ZMP Diagram").
*   **Placement**: Images must appear *after* the introductory hook but *before* the deep technical details.
*   **Sizing**: Max height of 350px to ensure they don't push content below the fold.
